# MERN-Shop-TailwindCSS
>MongoDB, Express, React, Node <br/>
>TailwindCSS Redux

### Install Dependencies

#### frontend
```
cd frontend
npm i
```
#### backend
```
npm i
```
### Start Project frontend

```
cd frontend
npm start
```
### Start Project backend

```
npm start
```
